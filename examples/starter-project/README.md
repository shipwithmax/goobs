# Example: starter-project

A pre-cooked end-to-end demo lands here at the end of phase 2. It'll include:

- a real `source.mp4` (one of Max's own videos, ~60s)
- the corresponding `0-plan/brief.md`
- a `1-log/transcript.json` and `1-log/cues.json` you can read
- a `2-assemble/rough.mp4` you can scrub
- a `3-enrich/enriched.mp4` showing the motion graphics
- final renders in `5-render/masters/`

This becomes the "wow" moment for first-time users — clone the repo, open this folder, watch what's possible.

A complete sample project will be added as the end-to-end pipeline is dogfooded.
