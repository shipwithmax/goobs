# Hangar

Your video projects live here. One folder per video.

After phase 2 ships, this README will auto-update with a table of every project and its current stage. For now: just inspect the folders directly.

## Folder convention

```
hangar/Y26-04-25-my-first-video/
├── source.mp4
├── 0-plan/
├── 1-log/
├── 2-assemble/
├── 3-enrich/
├── 4-polish/
├── 5-render/
└── 6-schedule/
```

## Naming convention

`Y<YY>-<MM>-<DD>-<slug>` — date prefix sorts naturally, slug is human-readable. Run `vp plan --topic "your topic"` and the folder name is generated for you.
